#E-commerce de placas de vídeo

Projeto front-end de um mini e-commerce inspirado no KaBuM, desenvolvido com foco em arquitetura, organização de código e boas práticas.

Funcionalidades
- Terá pagina home onde será o primeiro contato do cliente com interface.
- Listagem de produtos
- Busca por nome
- Carrinho de compras
- Persistência com localStorage
- Dados mockados

Arquitetura
O projeto utiliza separação de responsabilidades:
- `data/` → simulação de backend
- `services/` → regras de negócio
- `pages/` → lógica de páginas
- `utils/` → funções reutilizáveis

Tecnologias
- HTML5
- CSS3
- JavaScript (ES Modules)
- localStorage

Objetivo
Projeto desenvolvido para estudo e portfólio, simulando um fluxo real de desenvolvimento front-end utilizado no mercado.
